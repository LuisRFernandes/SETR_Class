Detailed Description

Testes usando Unit Testing para o cmd_processor.

Este ficheiro contém testes feitos ao cmd_processor para detetar possiveis erros de transmissão de dados, como por exemplo o envio de dados vazios ou com comprimento errado. Os testes são feitos com recurso à biblioteca Unity.
